## 开发原则
* 本仓库的风格指南仅适用于本人的个人项目及本人主导的项目。
* 每一个细节会根据所在团队和项目的变化而调整。
* 参与和维护其他人的项目时，秉承入乡随俗的原则，尊重别人代码的风格习惯。
